# TIC TAC TOE (JOGO DA VELHA) ONLINE. 

## Qual a idéia deste repositório? 
  ### Este Repositório tem como principal objetivo, não utilizar nenhuma IA (CHAT-GPT, COPILOT...) para me ajudar a criar este jogo,
  ### o foco é maximar o aprendizado em Comunicação BI-DIRECIONAL, algum Pattern orientado a eventos. E vivenciar algumas aventuras ao decorrer. 
## Quais técnologias irei utilizar? 
  ### Irei utilizar Typescript como linguagem, blibiotecas SOCKET.IO para emitir eventos, Express como framework já que será para WEB, creio que
  ### utilizarei REDIS como banco de dados, ou mongodb, e DOCKER para criar um ambiente easy de rodar, e algum serviço de Deploy como Azure ou Vercel. 
  
## Como rodar o projeto:
  ### Acesse o link do deploy, ou, copie o repositório e rode docker compose up na pasta em sua maquina -> (rodando somente o multiplayer local na sua maquina).
